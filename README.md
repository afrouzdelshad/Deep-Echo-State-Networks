# Deep Echo State Networks

The MATLAB implementation of this code was used to generate the results presented in the paper Predicting complex time series with deep echo state networks. The paper is available at DOI: 10.1063/5.0283425. If you use this code, please cite:
Delshad A, Cherry EM. Predicting complex time series with deep echo state networks. Chaos: An Interdisciplinary Journal of Nonlinear Science. 2025 Sep 1; 35(9). https://doi.org/10.1063/5.0283425
